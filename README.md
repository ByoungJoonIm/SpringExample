### TroubleShooting
- 파일명이 너무 길어 압축이 안풀리는 경우
  - C://로 압축파일을 옮긴 뒤 더블클릭 -> 우클릭, 현재 폴더에 압축 풀기 클릭
- Intellij에서 프로젝트의 패키지들이 한줄로 표시되는 경우
  - 프로젝트 익스프롤러의 우상단 톱니바퀴 클릭 -> Flatten Packages 체크, compact middle packages 체크 해제
